2. Core architecture rule

This rule is absolute:

LLMs never directly write to the system of record.

Agents may only produce structured JSON outputs.

All writes, external API calls, emails, policy binding, invoicing, and state changes must be performed by deterministic services after validation and approval.

Design explicit enforcement for this rule in code architecture, permissions, service boundaries, and queue flows.
